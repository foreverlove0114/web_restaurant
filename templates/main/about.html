Our Story

"Flask & Feats" is an educational project created by me in 2025. Its goal is to show how to implement a modern online restaurant using the Flask web framework.

This website was created as a learning project for practicing web development, improving interface design skills, and working with databases.

The project demonstrates the combination of technology and design: a user-friendly interface, responsive design, and functionality that allows ordering dishes online.
